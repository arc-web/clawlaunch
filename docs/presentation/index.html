Under the hood

How it works

📥
Run setup command on VPS
User executes the ClawLaunch script which prompts for Claude Max credentials and initiates OAuth flow.
⚙️
Generate 1-year subscription token
Tool exchanges Claude Max OAuth credentials for a long-lived token that stays valid for 12 months without re-authentication.
📤
OpenClaw runs on subscription
User gets a configured OpenClaw instance authenticated to Claude Max, eliminating per-token billing and staying TOS compliant.
